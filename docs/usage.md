## Quick Start

### Basic Download
```bash
# Download a single track
gamdl https://open.spotify.com/track/trackid

# Download entire playlist
gamdl https://open.spotify.com/playlist/playlistid

# Download album
gamdl https://open.spotify.com/album/albumid
